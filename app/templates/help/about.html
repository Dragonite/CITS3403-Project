About

  • Fishing Polling is made for polls related to fishing. It allows users to create polls about fishing, and gauge interest from other users on similar interests.

Account Types

  • Administrator - Administrators have the ability to add and delete polls, delete responses to polls and add and delete users.
  • Registered User - Registered users can view polls and current standings, and submit responses to polls.
  • Unregistered User - Unregistered users can view polls, and can also view current standings if the poll owner allows it.

Poll Types

  • Completed - Completed Polls refer to polls marked as completed. Votes can no longer be casted.
  • Active - Active Polls refer to current polls that are still in progress.
  • Inactive - Inactive Polls refer to polls that have been deleted by an Administrator. They are stored as a form of history.
